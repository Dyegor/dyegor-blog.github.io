Sprint 4 Tech

Got blocked on a simple problem

I was trying to implement a merge sort algorithm. The problem was that I couldn't imagine how all the elements should work together. I tried not to google the out-of-the-box solution because I wanted to solve it myself. After I solved the problem, I felt so fulfilled that it made me happy.

Solved a problem in an elegant way.

The link sent a request to the back end of the application, but the server did not receive any requests. To understand where the error occurred, i used console.log to check each step of the request being sent to the backend. When all the details were printed out, I discovered that it was a simple typo in the parameter name. After that i learn that it can take hours to find and fix even one single character misspeling, as well as how powerful the debugging technique is.
